/origin - Brings up a list of commands in game.

/origin enchant <enchantment> <level> - Enchants the current held item with a GenesisMc enchant.

/origin get <player name> - Returns the origin of the given player.

/origin give <player name> <origin item> <amount> - Gives a player an origins mod item.

/origin gui <player name> - Opens the origin choose menu for a given player.

/origin has <player name> <origin> - Checks if a player has a given origin.

/origin info - Opens a menu showing your origins powers.

/origin recipe - Opens a menu showing the crafting recipe for the origin of origins if it is enabled.

/origin references - Sends a list of credits in the chat.

/origin set <player name> <origin tag> - Sets the given player to a given origin.

/origin bug - Returns links in chat for places to report bugs to.
