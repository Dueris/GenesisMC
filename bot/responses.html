These are the commands that you can do with the origins mod! /origin choose - Opens the origins choosing menu if you have no origin /origin commands - Brings up a list of commands in game. /origin dump - Prints the config ino into the server console. /origin enchant <enchantment> <level> - Enchants the current held item with a GenesisMc enchant /origin texture - Enables custom GenesisMC texture pack /origin get <playername< - returns the origin of the given player /origin info get <args< - returns the values of the origin of the sender /origin purge <playername< - removes the origin of the given player /origin reload - reloads all config files. nope it doesn't work /origin gui <playername< - forces given player to rechoose their origin
